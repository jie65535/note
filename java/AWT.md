# java.awt

`AWT` 全称 `Abstract window toolkit` `抽象窗口工具包`，提供了一组丰富的平台无关的方式来创建图形用户界面的库。

本人并不关心其用于制作用户界面的部分，主要使用它的图形库进行图像渲染，因此笔记中不会记录窗口或者控件之类的内容，仅记录渲染功能。
